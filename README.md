# ft_transcendence
> This project is about creating a website for the mighty Pong contest!

## Table of Contents
* [Overview](#overview)
* [Features](#features)
* [Usage](#usage)
* [Demo](#Demo)


## Overview
- Users will play Pong with others.

## Features
- Use a Framework as backend, and that framework is Django .
- Database: PostgreSQL
- Frontend : Javascript, Bootstrap, CSS
- Remote players: 1 vs 1, 2 vs 2, Tournament It is possible to have two distant players. Each player is located on a separated computer, accessing the same website and playing the same Pong game.
- Two-Factor Authentication (2FA) and JWT, OAuth


## Usage
in terminal, clone:
```
git clone https://github.com/JosepharDev/ft_transcendence.git && cd ft_transcendence
```
run:
```
docker-compose up --build
```

 ```
 https://127.0.0.1:8000
 ```
